Summary

The purpose of this assignment was to analyze how different variables associated with weather change as you get closer to the equator. More specifically this was a comparison study that compared different cities latitude with factors such as maximum temperature, wind speeds, cloudiness and humidity. To accomplish this task, data from the OpenWeatherMap API was used to gather information on 500 random cities.

After the data set was properly rendered, the Python library Matplotlib was used to plot the various visualizations. This site provides the source data as well as the visualizations created as part of the overall analysis. Explanations and descriptions of any trends observed are also included. I hope this is a helpful tool for all viewers! Enjoy!
